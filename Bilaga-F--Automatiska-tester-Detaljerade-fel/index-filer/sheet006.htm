1584858 bild1 3 - Medium 1.1.1  https://utbildning.ki.se/alla-program <img> element utan alt-attribut Elementet <img> ska alltid ha attributet alt i sig. För bilder som inte är meningsbärande ska alt-attributet vara tomt, men själva attributet ska alltid finnas i elementet. För hjälpmedlen utgör bilder utan alt-attributet ett problem eftersom hjälpmedlet då inte vet om det är en meningsbärande bild eller inte. Följden kan bli att hjälpmedlet återger annan information, exempelvis bildens sökväg, i ett försök att ge användaren den information som krävs för att förstå sidan och funktionerna. Komplettera alla <img> element med alt="".
